# コマ3: モデル学習（1回目）（1.5時間）

## 🎯 このコマの目標

1. 全データセットで本格的にBERT学習を実行する
2. 学習過程を監視する
3. 初期精度を確認する
4. 学習ログを記録する

## 📋 タスクリスト

### タスク1: 学習実行前の確認（10分）
- [ ] データセットの最終確認
- [ ] パラメータの最終確認
- [ ] ディスク容量の確認
- [ ] GPU/CPUメモリの確認

### タスク2: 学習実行（50-70分）
- [ ] `02_train_bert.py` を実行
- [ ] 学習過程を監視
- [ ] Lossの推移を確認
- [ ] 異常がないか確認

### タスク3: 結果の確認（15分）
- [ ] Test Accuracyの確認
- [ ] F1スコアの確認
- [ ] 混同行列の分析
- [ ] 誤分類サンプルの抽出

### タスク4: 結果の記録（10分）
- [ ] スクリーンショットの保存
- [ ] 結果のまとめ作成
- [ ] 改善ポイントの洗い出し

### タスク5: 次のコマへの準備（5分）
- [ ] エラー分析の方向性決定
- [ ] 改善案のリストアップ

## ✅ 完了条件

- [ ] 学習が正常に完了した
- [ ] Test Accuracy 70%以上を達成
- [ ] `training_results.json` が生成された
- [ ] `first_training_report.md` が作成された
- [ ] 誤分類サンプルが抽出された

## 📂 成果物
