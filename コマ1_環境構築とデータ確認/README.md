# コマ1: 環境構築とデータ確認（1.5時間）

## 🎯 このコマの目標

1. 必要なライブラリをインストールして開発環境を整える
2. データセットを読み込んで正常性を確認する
3. 基本的なEDA（探索的データ分析）を実施する
4. 次のコマ（モデル学習）への準備を完了する

## 📋 タスクリスト

### タスク1: 環境構築（20分）
- [ ] Python環境の確認（Python 3.8以上）
- [ ] 必要ライブラリのインストール
- [ ] GPUが使用可能か確認

### タスク2: データ読み込みテスト（15分）
- [ ] train.jsonl, val.jsonl, test.jsonlの読み込み
- [ ] データ構造の確認
- [ ] 欠損値・異常値のチェック

### タスク3: 基本的なEDA（30分）
- [ ] ラベル分布の可視化
- [ ] 文字数分布の分析
- [ ] 頻出語の抽出
- [ ] サンプルデータの詳細確認

### タスク4: トークナイズテスト（20分）
- [ ] BERTトークナイザーの動作確認
- [ ] サンプルテキストのトークナイズ
- [ ] トークン長の分布確認

### タスク5: レポート作成（5分）
- [ ] 分析結果のサマリー作成
- [ ] 次のコマへの引き継ぎ事項整理

## ✅ 完了条件

- [ ] すべての必要ライブラリがインストール済み
- [ ] データセットが正常に読み込める
- [ ] EDAスクリプトが実行でき、結果が可視化できる
- [ ] `eda_report.md` が作成されている
- [ ] 次のコマへの準備が整っている

## 📂 成果物

- `requirements.txt` - 依存ライブラリリスト
- `01_eda.py` - EDAスクリプト
- `eda_report.md` - 分析結果レポート
- `figures/` - 可視化結果の画像フォルダ

## ⏭️ 次のコマへの引き継ぎ

- データセットの特性（平均文字数、トークン長など）
- 推奨されるmax_length設定
- 特に注意が必要なデータの特徴
