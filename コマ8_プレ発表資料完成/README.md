# コマ8: プレ発表資料完成とリハーサル（1.5時間）

## 🎯 このコマの目標

1. プレ発表資料を最終調整する
2. 複数回リハーサルを行う
3. 質疑応答の練習を行う
4. 自信を持って発表できる状態にする

## 📋 タスクリスト

### タスク1: 資料の最終調整（20分）
- [ ] スライドの修正
- [ ] 誤字脱字チェック
- [ ] 図表の最終確認
- [ ] 発表ノートの見直し

### タスク2: リハーサル（50分）
- [ ] 1回目: 通し練習（時間計測）
- [ ] フィードバック・修正
- [ ] 2回目: 改善点を意識して練習
- [ ] 3回目: 本番想定で練習

### タスク3: 質疑応答練習（15分）
- [ ] 想定質問への回答練習
- [ ] 回答の言い方を練習
- [ ] 予想外の質問への対応練習

### タスク4: 最終確認（5分）
- [ ] すべての資料が揃っているか確認
- [ ] デモの動作確認
- [ ] 機材・接続確認（必要に応じて）

## ✅ 完了条件

- [ ] スライドが完璧な状態
- [ ] 3回以上リハーサルを実施
- [ ] 5分以内に安定して発表できる
- [ ] 質疑応答に自信がある
- [ ] プレ発表の準備が完全に整った

## 📂 成果物

- `プレ発表スライド_最終版.pptx` - 最終版スライド
- `発表チェックリスト.md` - 本番前確認事項
- `リハーサル記録.md` - 練習の記録

## 🎯 リハーサルの目標

### 1回目
- 時間配分を確認
- 詰まる箇所を特定

### 2回目
- スムーズさを向上
- 改善点を意識

### 3回目
- 本番想定で完璧に
- 自信を持って話す

## ⏭️ プレ発表本番へ

- 完璧に準備された状態
- 自信を持って臨める
- 楽しんで発表する
